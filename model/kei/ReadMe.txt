
============================================================

    샘플 모델
    케이

============================================================

    리얼한 립싱크를 구현할 수 있는 모션 싱크의 기능을 배울 수 있는 모델입니다.
    립싱크의 기본 프리셋에 대응하는 파라미터 구성이므로 입을 만드는 방법에 참고가 됩니다.
    프리셋마다 모델 파일을 준비하고 있습니다.
    포함된 오디오 파일로 모션 싱크 기능을 사용해 볼 수 있습니다.


------------------------------
    라이선스에 대해서
------------------------------

    일반 사용자 및 소규모 사업자는 규약 동의 후 상업적 이용 가능.
    중, 대규모 사업자는 비공개 테스트 용도로만 이용 가능.
    이용 시에는 아래의 "무상 제공 소재의 이용 허락 계약서"
    (라이선스 타입 · Live2D 오리지널 캐릭터) 및 이용 조건에 대한 동의가 필요합니다.

    라이선스에 대한 자세한 내용은 아래 페이지를 참조하시기 바랍니다.
    https://www.live2d.com/ko/download/sample-data/

------------------------------
    샘플 데이터 구성
------------------------------

    - 모델 데이터 (.cmo3)
    - 기본모션 (.can3)
    - 삽입용 파일일식 (runtime 폴더)
        - 모델 데이터 (.moc3)
        - 모션 데이터 (.motion3.json)
        - 모델 설정 파일 (.model3.json)
        - 음성 파일 (.wav)
        - 물리 연산 설정 파일 (.physics3. json)
        - 모션 싱크 설정 파일 (.motionsync3.json)
        - 표시보조파일 (.cdi3.json)
